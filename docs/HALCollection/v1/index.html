HALCollection Specification

  • Media Type: application/hal+json
  • Profile: https://arucard21.github.io/SimplyRESTful/HalCollection/v1
  • Version: 1

Attributes

  • page: is the number of the page that's currently shown from the collection
  • pageSize: is the maximum amount of resources that are being shown on a single page of the collection
  • total: is the number of resources that are available in the entire collection

Embedded

  • item: is a list of the resources from this page, where the entire resource is shown for each. This is only visible when the compact query string parameter is set to false.

Links

  • item: is a list of links to the resources that are part of this page of the collection. This is only visible when the compact query string parameter is omitted or set to true.
  • self: is a link to the collection itself.
  • first: is a link to the first page of the collection.
  • last: is a link to the last page of the collection.
  • prev: is a link to the next page of the collection.
  • next: is a link to the next page of the collection.

Search

Using the _s or _search query string parameter, you can search the resources in the collection. This can be done using the Feed Item Query Language (FIQL) where you can filter on any of the resource's attributes.
